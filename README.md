#  Tomáš Urban

## 🎥 About 
This tool creates VFX for films.
It has a wide range of applications and is best used collaboratively.

> Currently, under development by [this institution](https://www.digitalartsandentertainment.be/) 🎓


## 💾 Features

- Creates pipeline tools
- Models in Blender
- Composites in Fusion
- Simulates in Houdini
- Shoots films


## 💡 Known issues
 
- Occasionally runs out of memory
- Has terrible facial recognition
- Database is limited to 1940s jazz songs
- Always smiles. Always.
